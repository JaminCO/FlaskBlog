CHIRPERR API endpoints

Chirperr API endpoints and its uses.

Endpoints Endpoint Uses
/api/post/post_id This is used to only get the post that belongs to the id,
/api/post/all This is used to get all the post in chirperr, it is only used to get the post id and then use it in the above endpoint
/api/memebership This is used to get the membership status of the current user
/api/membership/(username) This is used to get the membership status of the specified user
/api/login This is used for logging into chirperr without going to the login page, follow the instructions given or use the below endpoint
/api/login/email=(youremail) &password=(yourpassword) This is used for logging into chirperr without going to the login page
/api/post/new This is used for posting in chirperr without going to the website, follow the instructions given or use the below endpoint
/api/post/new/(yourapikey)/ title=(yourtitle)&content=(yourcontent) This is the api endpoint for posting
/api/account This is used for getting the account information for the logged in user. It returns it in JSON Format
/api/(yourapikey)/account/ (username_of_accounts_user) This is used for getting the account information for the specified user. It returns it in JSON Format
