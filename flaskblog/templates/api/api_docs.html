API LOGIN ENDPOINT

URL: http://127.1.1.1:5000/api/login/email=youremail&password=yourpassword

Enter your login in details like shown below

Example login: http://127.1.1.1:5000/api/login/email=test@chirperr.com&password=testing

API NEW POST ENDPOINT

URL: http://127.1.1.1:5000/api/post/new/(yourapikey)/title=(yourtitle)&content=(yourcontent)

Enter your login in details like shown below

Example new_post: http://127.1.1.1:5000/api/post/new/ab73c-62he3-su1s4/title=MY TITLE&content=My content

API ACCOUNT ENDPOINT

Enter your login in details like shown below

Example new_post: http://127.1.1.1:5000/api/ab73c-62he3-su1s4/account/Testuser
